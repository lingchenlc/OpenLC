### 🚀 Capability Evolver - 顶级自我进化 Skill

**下载量**: 35,581 (Claw Hub 第一)

**功能**:
- **Auto-Log Analysis**: 自动扫描 memory 和 history 文件中的错误和模式
- **Self-Repair**: 检测崩溃并建议补丁
- **GEP Protocol**: 标准化的可审计进化协议
- **One-Command Evolution**: 运行 `/evolve` 即可

**使用方式**:
```bash
# 标准自动运行
node index.js

# 审核模式 (人类确认后再应用)
node index.js --review

# 连续循环模式
node index.js --loop
```

**环境变量配置**:
| 变量 | 默认值 | 说明 |
|------|--------|------|
| `EVOLVE_ALLOW_SELF_MODIFY` | false | 允许修改自身代码 (生产环境不推荐) |
| `EVOLVE_LOAD_MAX` | 2.0 | 负载阈值，超过则退避 |
| `EVOLVE_STRATEGY` | balanced | 策略: balanced/innovate/harden/repair-only |

**GEP Protocol** (可审计进化):
- `genes.json`: 可复用基因定义
- `capsules.json`: 成功胶囊 (避免重复推理)
- `events.jsonl`: 进化事件日志

**安全特性**:
- Review Mode 保护
- Git sync 建议配合使用
- 风险缓解: 无限递归保护

### OpenClaw Memory 系统 (2026 官方文档)

**核心原则**: "plain Markdown in the agent workspace" - 文件是唯一真相

**双层记忆**:
- `memory/YYYY-MM-DD.md` - 每日日志 (append-only)
- `MEMORY.md` - 精选长期记忆 (仅主session加载)

**工具**:
- `memory_search` - 语义检索
- `memory_get` - 精准读取

**自动 Memory Flush**:
- Session 接近 compaction 时自动触发
- 静默写入关键信息到 memory 文件
- 防止上下文压缩导致记忆丢失

**相关 Skills**:
- `claw-progressive-memory`: 元技能
- `claw-roam`: 多机器同步 workspace
- `context-anchor`: 从 memory 文件恢复上下文
