## 2026-03-01 AI Agent 自我进化与亚马逊广告研究摘要

### 1. AI Agent 自我进化 (Self-Evolution)
- **openclaw-inner-life**: 为 OpenClaw 提供的模块化插件（evolve, reflect, memory），允许 Agent 记录并优化行为。
- **Geneclaw**: 具备 5 层安全门控的框架，支持 Agent 观察失败并安全修复。
- **agent-evolver**: 专注于从经验中检测问题并提取洞察的进化引擎。

### 2. OpenClaw 技能开发最佳实践
- **ClawHub 声明**: 推荐使用 metadata 标识。
- **精简原则**: 描述需精简，减少冗余，使用分级控制以降低风险。
- **资源库**: awesome-openclaw-skills 包含海量分类技能可供参考。

### 3. 亚马逊广告 (Amazon Ads) 自动化
- **Amazon Ads MCP Server**: 亚马逊官方推出的 MCP 协议服务端，是 Agent 连接广告 API 的桥梁。
- **自动化任务**: 支持 Agent 自动创建活动、调整预算、管理产品及拉取报告。
- **官方助手**: Ads Agent 已上线，用于全自动化的投放优化和建议生成。
