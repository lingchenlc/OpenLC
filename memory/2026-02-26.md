### OpenClaw Memory 问题修复指南
📘 [Markaicode](https://markaicode.com/fix-openclaw-memory-issues/)

**三大失败模式**:
1. **Session 间失忆** - 文件从未写入 MEMORY.md
2. **对话中失忆** - context compaction 未保存到内存
3. **手动 /reset** - 丢弃 session 未保存

**修复步骤**:
1. 验证 memory 文件存在 (MEMORY.md, memory/YYYY-MM-DD.md)
2. 启用自动 memory flush:
```json
"compaction": {
  "memoryFlush": {
    "enabled": true,
    "softThresholdTokens": 4000
  }
}
```
3. 配置 hybrid search (vector 0.7 + BM25 0.3)
